# VAE_music_score
Implementing a simple VAE on musical scores to specific dimensions.
How to run.
1) Clone repository.
2) Run "run_data_extraction.py" to make data.
3) Run "music_autoencoder.py".
4) See results/examples in figs.

See comments in music_autoencoder.py.
